Mopidy extension for parsing FIP current playing title


Installation
============

Install by running::

    pip install Mopidy-FipNowPlays


Configuration
=============

Before starting Mopidy, you must add configuration for
Mopidy-FipNowPlays to your Mopidy configuration file::
